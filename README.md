# 智能看图配文系统
## 📖 项目介绍 (Introduction)
针对社交媒体(如朋友圈、TikTok类应用)的配文需求，开发了一套结合图像识别与大模型文本生成的系统，支持网页端与小程序双端访问。
系统核心流程：
1.  **图像理解**：利用ClipCap模型对上传的图片进行深层特征提取，生成英文描述。
2.  **文本生成**：将英文描述与用户的情感倾向（开心/忧伤/平静）拼接，构建Prompt。
3.  **润色输出**：调用百度文心一言大模型接口，生成符合社交语境的中文文案（支持 Emoji）。
